I'm going to pretend my name is Spiderman for this. Everywhere you see Spiderman, just replace with your info

View all git config stuff
  spiderman@computer$ git config --list
  spiderman@computer$ git config -l

Set username
  spiderman@computer$ git config --global user.name "spiderman"

Set email
  spiderman@computer$ git config --global user.email "spiderman@web.com"
